Техника

Разверните образовательную платформу за 1 минуту

Технически процесс одинаков для любого проекта. Далее вы наполняете его своим курсом и правилами форума.

  1. 1

    Регистрация на GitHub

    Создайте бесплатный аккаунт на github.com (2 минуты)

  2. 2

    Откройте базовый шаблон

    github.com/feekool/feekool.github.io

  3. 3

    Скопируйте в свой репозиторий

    Нажмите Use this template → Create a new repository. Имя: nutri-course-forum

  4. 4

    Включите публикацию

    Settings → Pages → ветка main → Save. Сайт появится по адресу вашник.github.io/репозиторий

  5. 5

    Настройте под обучение

    Замените заголовки, добавьте разделы «Модули», «Домашки», «Вопросы ментору». Загрузите PDF-материалы. Готово.

💡 Для образовательных проектов: Начните с 2 разделов — «Теория» и «Обсуждение». По мере набора студентов добавляйте «Проверка заданий» и «Закрытый клуб выпускников».
